# BossFightRPG - 召唤·搬金·Boss战 3D Game Specification (v3)

## 1. 游戏概述

### 核心概念

玩家操控一个英雄角色，在基地与Boss战场之间往返。攻击Boss使金块掉落在地上，英雄拾取金块背在背上运回基地，在基地的升级站消耗金块来升级武器或孵化龙蛋精灵，变得更强后继续挑战更强的Boss。

### 核心循环

```text
操控英雄 → 走向Boss → 自动攻击Boss → 金块掉落地面 → 
捡起金块(堆在背上) → 跑回基地 → 在升级站卸下金块 → 
升级武器/孵化龙蛋 → 变强 → 继续打Boss → 击杀Boss → 下一关
```

### 核心卖点

- 操控英雄的参与感
- 背着金块跑回基地的搬运爽感
- 武器升级的即时战力提升
- 龙蛋孵化出精灵龙的惊喜感
- 巨型Boss的视觉冲击力

### 一句话描述

操控英雄打Boss、捡金块、背回基地升级武器和孵龙,越来越强直到击杀所有Boss。

### 玩家英雄

| 属性 | 描述 |
| --- | --- |
| 外观 | 弓箭手/战士，蓝色铠甲，手持弓 |
| 移动速度 | 8 单位/秒 |

### 精灵龙(Dragon Pet)

> **每只龙比前一只更强、更大、攻击特效更华丽**

| 龙 | 外观 | 攻击力 | 特殊能力 |
| --- | --- | --- | --- |
| 第1只: 火焰幼龙 | 小型红色龙，翼展小 | 60 DPS | 发射单发小火球 |

- 每只龙跟随英雄飞行，自动攻击Boss
- 龙的体型逐渐增大，视觉上越来越震撼
- 最多 3 只精灵龙同时作战

### Boss AI 行为

| 行为 | 描述 |
| --- | --- |
| 待机动画 | Boss 缓慢左右转身 + 身体轻微上下浮动 |
| 移动 | Boss 会缓慢向英雄方向移动(速度 2 单位/秒) |
| 转身 | Boss 始终面向英雄方向 |
| 普通攻击 | 每 3 秒一次近战攻击动画(身体前倾冲击)，造成轻微伤害 |
| 技能1: 地面砸击 | 每 8 秒，Boss 跳起砸地，产生扩散冲击波环，造成范围伤害 |
| 技能2: 冲锋 | 每 15 秒，Boss 向英雄方向快速冲刺一段距离 |
| Boss受伤反馈 | 身体闪红 0.1秒 |
| 伤害数值 | Boss攻击造成 5-10 伤害(保证爽感，玩家HP不会低于10%) |
| Boss领地 | 每个Boss有10单位半径的巢穴区域，Boss不会追出巢穴进入主城 |

```text

    |

    |
    ═══[北 门]═══

    │        [英雄出生点]        │ ← 基地中央
```

> **重要**: 基地面积 20×16 单位。Boss 距离基地约20单位，确保搬运金块的往返不会太久。基地有南北两扇门，Boss交替在不同方向刷新。城墙不可穿透，玩家必须走门进出。

### 升级站

| 升级站 | 外观 | 功能 | 位置 |
| --- | --- | --- | --- |
| 武器升级台 (⚔️) | 铁砧 + 方形金色触发区域 | 站入区域自动消耗金币升级 | 基地左侧(x=-6) |
| 龙蛋孵化台 (🥚) | 龙巢+发光龙蛋 + 方形金色触发区域 | 站入区域自动消耗金币孵龙 | 基地右侧(x=6) |

**购买区域机制**:
- 购买区域为 **正方形**(半边长2.2单位)，地面显示金色边框
- 玩家走入区域后自动消耗金币(每0.12秒消耗1个)
- 区域内显示 **矩形进度条**(从下往上填充)
- 地面大字显示金币进度(如 "🪙 3/5")
- 达到要求后自动升级，显示 "⬆ 等级提升，获得 XXX！"

## 5. 核心系统

### 5.1 操控系统

| 操作 | PC端 | 移动端 |
| --- | --- | --- |
| 移动 | WASD 键盘 或 鼠标拖拽(摇杆跟随鼠标) | 虚拟摇杆(跟随触摸位置) |
| 攻击 | 自动(接近Boss即攻击) | 自动 |
| 拾取金块 | 自动(走过即拾取) | 自动 |

**摇杆规则**:
- PC端：鼠标按下时，在鼠标按下位置生成摇杆底座，拖拽方向控制英雄移动，松开后摇杆消失
- 移动端：触摸时在触摸位置生成摇杆底座，拖拽方向控制英雄移动
- WASD 始终可用

**教程引导(游戏开始前)**:
- 游戏开始时处于 `idle` 状态，英雄站在原地
- 屏幕上显示一个半透明的 **手指动画**，在摇杆区域做圆形运动
- 手指旁边显示文字 "Drag to move!"
- 玩家首次操作(鼠标按下/触摸/WASD)后，教程手指消失，游戏正式开始

### 5.2 战斗系统

| 参数 | 值 |
| --- | --- |
| 攻击范围 | **5 单位**(远程弓箭射击) |
| 攻击频率 | 1 次/秒 |
| 每次攻击伤害 | 英雄攻击力值 |
| Boss 行为 | Boss 主动移动+转身面向英雄，有攻击和技能(详见 §3 Boss AI) |
| 金块掉落 | 每次攻击 Boss 掉落 3 个金块，**散落在 Boss 周围半径 2-4 范围**，集中在Boss附近 |
| 击杀Boss | Boss HP=0 时先消失，再爆出大量金块(分批掉落，更有节奏感) |
| Boss出现提示 | 屏幕中央大字显示 "⚠️ [Boss名] 已出现！快去消灭它！" (3秒后消失) |

### 5.3 金块搬运系统

| 参数 | 值 |
| --- | --- |
| 拾取方式 | 英雄走到金块附近(半径 2)自动拾取 |
| 背负上限 | **无上限**，可背无限多金块 |
| 视觉表现 | 金块堆叠在英雄背上，越多堆越高(竖向排列) |
| 卸载方式 | 英雄走到升级站附近(半径 3)自动卸下金块 |
| 卸载效果 | 金块逐个飞向升级站，升级站金块计数增加 |

### 5.4 升级系统

#### 弓箭升级台 (🏹 Bow Forge)

> **每次升级都从本质上提升弓箭威力，攻击特效大幅加强**

| 等级 | 消耗金块 | 弓箭形态 | 攻击力 | 攻击特效 |
| --- | --- | --- | --- | --- |
| Lv1 (初始) | - | 普通木弓，单支箭矢 | 100 DPS | 发射 1 支白色箭矢，直线飞行 |
| Lv2 (三连射) | 5 金块 | 精钢弓，弓臂发蓝光 | 300 DPS | 同时发射 **3 支箭矢**(扇形散射)，蓝色拖尾 |
| Lv3 (魔法箭) | 10 金块 | 魔法弓，整体发紫色光芒 | 600 DPS | 发射 **3 支紫色魔法箭**，命中后产生 **紫色魔法爆炸** 范围伤害 |
| Lv4 (追踪箭暴) | 15 金块 | 传说弓，金色发光+粒子环绕 | 1200 DPS | 发射 **5 支金色追踪箭**，箭矢自动追踪Boss，命中产生 **金色爆炸** |

- 每次升级时：金色光柱特效 + 弓箭外观明显变化
- 箭矢颜色随升级变化：白色 → 蓝色 → 紫色 → 金色
- 弓的大小和发光强度逐级增加
- 升级瞬间有 "LEVEL UP!" 文字弹出

#### 龙蛋孵化台 (🥚 Dragon Nest)

> **龙蛋需从Boss处获得，每只Boss掉落一枚龙蛋**

**孵化台建造与使用流程：**

1. **建造阶段** — 孵化台区域初始为空地，玩家捐献 8 金币后孵化台(巢穴)出现
2. **等蛋阶段** — 孵化台显示"击败Boss获得龙蛋"，击败Boss后龙蛋掉落在Boss位置
3. **拾取龙蛋** — 玩家走近龙蛋自动拾取，龙蛋会显示在玩家头顶
4. **放置龙蛋** — 玩家走入孵化台区域，龙蛋自动放置到巢穴上
5. **孵化阶段** — 继续捐献金币，进度条填满后幼龙孵化
6. **重复** — 孵化台再次进入等蛋阶段，等待下一个Boss掉落龙蛋

| 龙蛋 | 来源 | 孵化费用 | 孵出 | DPS | 攻击特效 |
| --- | --- | --- | --- | --- | --- |
| 第1枚 (红蛋) | Boss1 夜王 | 8 金块 | 🔴 火焰幼龙 | 60 | 发射小火球 |
| 第2枚 (紫蛋) | Boss2 赤焰巨龙 | 15 金块 | 🟣 雷电青龙 | 120 | 发射闪电弹 |
| 第3枚 (金蛋) | Boss3 双头巨熊 | 20 金块 | 🟡 黄金神龙 | 200 | 发射追踪火焰弹 |

- 孵化台建造费用: 8 金块
- 每只Boss固定掉落对应颜色的龙蛋
- 龙蛋颜色从红→紫→金，暗示递增的珍稀度
- 精灵龙跟随英雄飞行，同时攻击Boss
- 3只龙全出后总计 380 DPS 的额外火力

## 12. 关键体验原则

1. **操控感** — 玩家始终在操控英雄移动，不是被动观看
2. **搬运爽感** — 背上堆满金块跑回基地的满载感
3. **即时回馈** — 每次升级都能明显感受到伤害提升
4. **龙蛋惊喜** — 孵化精灵龙的视觉奖励
5. **Boss 包装** — 夸张体型 + 命名 + 等级 = 高级感
6. **节奏紧凑** — 打Boss → 搬金 → 升级 → 更强 → 循环不断
